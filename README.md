# Data Analytics — Portafolio

Repositorio con mis proyectos de análisis de datos, donde aplico Python, SQL y Power BI
sobre preguntas de negocio reales, usando datos públicos y financieros.

## Proyectos

### [¿Es viable que Sony elimine los juegos físicos para 2028?](./sony-physical-vs-digital)
Análisis exploratorio sobre el anuncio de Sony de dejar de fabricar discos físicos de
PlayStation, usando sus informes financieros públicos y datos filtrados de Insomniac Games.
Incluye tendencias históricas, desglose de ingresos y un contraargumento basado en el
rendimiento de sus exclusivos triple A.

**Herramientas:** Python · pandas · matplotlib · Jupyter Notebook
